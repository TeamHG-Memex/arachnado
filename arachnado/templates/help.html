    Help

    Arachnado is a tool to get all data from a website and store it in a database.

    Starting a Crawl

    Enter a website URL in the crawl form and click "Crawl" to start crawling. You can configure crawl options by clicking the gear icon.

    Uploading Custom Spiders

    You can upload your own Scrapy projects to run custom spiders:

    1. Package your Scrapy project as a .zip or .tar.gz file
    2. Click on "Upload Scrapy Project" panel on the main page
    3. Enter a project name (letters, numbers, underscores, and hyphens only)
    4. Select your project archive file
    5. Click "Upload Project"

    After uploading, you can run spiders from your project by entering spider://<spider_name> in the crawl form, where <spider_name> is the name of your spider.

    Managing Jobs

    Active crawl jobs are listed in the "Jobs" panel. You can stop, pause, or resume jobs using the controls shown for each job.
